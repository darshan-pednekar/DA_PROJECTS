# Google Ads Analysis Project

## Project Overview

This project analyzes Google Ads campaign data to evaluate advertising performance across campaigns, regions, customers, and keywords. SQL is used for data cleaning and analysis, and the results are visualized using Power BI dashboards.

## Objectives

Analyze overall Google Ads campaign performance

Understand campaign effectiveness across regions and customers

Evaluate keyword and time-based performance trends

Convert raw advertising data into actionable insights

## Dataset

The project uses Google Ads data stored in CSV format.

### Raw Data

Campaign performance data

Ads, campaigns, customers, keywords, and regions information

### Cleaned Data

Fact and dimension tables prepared for analysis and reporting

## Tools & Technologies

SQL

Power BI

CSV files

## Analysis Performed

Data cleaning and transformation using SQL

Overall performance metrics analysis

Campaign-wise and regional performance analysis

Customer segment analysis

Keyword performance analysis

Daily and monthly trend analysis

## Power BI Dashboard

The Power BI dashboard includes:

Overall performance metrics

Campaign performance summary

Regional and customer performance

Keyword analysis

Time-based trends

### Dashboard preview available in the Screenshot folder.

## Project Conclusion

This project helps in understanding how Google Ads campaigns perform using real advertising data. By analyzing campaign, keyword, and regional performance, useful insights were generated. The project shows how SQL and Power BI can be used together to analyze and visualize marketing data.
