# IMDB Movie Analysis Project

## Project Overview

This project analyzes IMDb movie data using SQL and Power BI to uncover insights related to movie ratings, directors, genres, and overall trends. The goal is to transform raw movie data into meaningful insights through querying and interactive dashboards.

## Objectives

Analyze IMDb movie ratings and movie details

Identify top-rated movies and directors

Perform data analysis using SQL queries

Build an interactive Power BI dashboard for visualization

## Dataset

The project uses IMDb movie datasets in CSV format:

Movies.csv – Movie details such as title, year, rating, votes, duration

Directors.csv – Director-related information

## Tools & Technologies

SQL – Data querying and analysis

Power BI – Data visualization and dashboard creation

CSV – Dataset format

## Analysis Performed

Data exploration using SQL queries

Movie rating analysis

Director-wise performance analysis

Identification of top-rated movies

Trend analysis based on available data

## Power BI Dashboard

The Power BI dashboard provides interactive visualizations including:

Top-rated movies

Rating distribution

Director-wise movie analysis

Key performance insights

#### Dashboard preview available in the screenshot folder.

## How to Use

Open the SQL file (imdb_movies.sql) to review analysis queries

Load CSV files into Power BI or your SQL environment

Open the Power BI .pbix file to explore the dashboard

Use filters and visuals to gain insights

## Conclusion

This project helped in understanding IMDb movie data by using SQL for analysis and Power BI for visualization. The analysis shows how movie ratings and director details can be used to identify popular and high-rated movies. Overall, the project demonstrates basic data analysis and dashboard creation skills using real-world data.
