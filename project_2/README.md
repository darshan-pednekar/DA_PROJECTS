# Test Cricket Analysis Project

## Project Overview

This project focuses on analyzing Test cricket match data to understand player and match performance using Python. The analysis is done in a Jupyter Notebook using real cricket data stored in CSV format.

## Objectives

Explore Test cricket match data

Perform data cleaning and preparation

Analyze performance patterns using statistical methods

Visualize insights from the dataset

## Dataset

Test Cricket.csv

Contains Test cricket-related data such as match details and performance metrics.

## Tools & Technologies

Python

Jupyter Notebook

Pandas

NumPy

Matplotlib / Seaborn

## Analysis Performed

Loaded and cleaned Test cricket dataset

Performed exploratory data analysis (EDA)

Analyzed performance trends using Python

Created visualizations to represent insights clearly



## Project Conclusion

This project helps in understanding Test cricket data using Python. By analyzing the dataset, useful patterns and insights were identified. It also shows how Python and Jupyter Notebook can be used for basic data analysis and visualization.
